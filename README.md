## 🌟 Featured Projects

- 🔁 **[Real-time RAG Engine](https://github.com/utk1725/realtime-rag)**  
  A live Retrieval-Augmented Generation pipeline using FAISS, SentenceTransformers, and Flan-T5. Supports live chat ingestion (Slack), fuzzy fallback, reranking with CrossEncoder, and real-time indexing.

- 🌱 **[Chickpea Anomaly Detection](https://github.com/utk1725/Chickpea-Anomaly-)**  
  An image-based ML model that detects anomalies in chickpea crops. Combines preprocessing, image augmentation, and classification.

- 🏦 **[Credit Approval System](https://github.com/utk1725/Credit-Approval-System)**  
  Predicts credit card approvals using Decision Trees, Logistic Regression, and Scikit-learn pipelines. Designed for fintech/data science evaluations.

- 🛡️ **[MERN Stack Auth System](https://github.com/utk1725/MERN-Stack-Authentication-System)**  
  Full authentication stack with JWT, secure routes, token handling, and role-based access — built using React, Node, Express & MongoDB.

- 🩺 **[Symptom Checker API](https://github.com/utk1725/SYMPTOM-CHECKER-API)**  
  A FastAPI-based medical NLP API that takes symptoms and maps them to possible diseases using dataset-driven matching and condition ranking.
